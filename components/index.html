Google Button

<GoogleButton appId="my app id" theme="dark" />

API

Default class(es) of the button:


Props Description Type Values Default Value
appId The id of your Discord app string - -
scopes Scopes for your app, seperated with a comma, i.e "profile, email" string profile, email profile
width Width of the button integer - 140
height Height of the button integer - 40
longTitle Display long labels such as "Sign in with Google" rather than "Sign in" boolean true, false false
theme Theme of button string light, dark light
onLoginSuccess A function if the login is a success, take the GoogleUser in paramater Function - () => {console.log('Logged to google with success!')}
onLoginFailure A function if the login is a failure, take the error in paramater Function - () => {console.log('Logged to google with failure!')}

Facebook Button

<FacebookButton appId="my app id" />

API

Default class(es) of the button:


Props Description Type Values Default Value
appId The id of your Discord app string - -
scopes Scopes for your app, seperated with a comma, i.e "public_profile, email" string public_profile, email, groups_access_member_info, publish_to_groups, user_age_range, user_birthday, user_events, user_friends, user_gender, user_hometown, user_likes, user_link, user_location, user_photos, user_posts, user_tagged_places, user_videos public_profile
type Type of button string login_with, continue_with login_with
audience Choose the default audience selected when you request write permissions string friends, only_me, everyone friends
size Size of button string small, medium, large large
language Language of the user, i.e en_US string - en_US
useContinueAs If user is already connected, show him a "continue as" button instead of "connexion"? boolean true, false false
autoLogoutButton If user is already connected to your app, show him a "Logout" button? boolean true, false true
debug Enable debug mode of Facebook's sdk boolean true, false false
onLoginSuccess A function if the login is a success, take the response in paramater Function - () => {console.log("Logged to facebook with success!")}
onLoginFailure A function if the login is a failure, take the response in paramater Function - () => {console.log("Logged to facebook with failure!")}
onStatusChange A function if the status of the user change Function - () => {console.log("Facebook status changed!")}
